# Отключить кэш без авторизации

---

Отключаем кэширование Битрикс без авторизации под админом.\
Установка этого значения отключит кэширование на время текущей сессии пользователя:

```php
$_SESSION['SESS_CLEAR_CACHE'] = 'Y';
```
